How to Apply CSS to HTML

To include an external CSS file in an HTML document, use the <link> element within the <head> section. Here's an example: 

  <link rel="stylesheet" href="style.css">
This line of code tells the browser to load the stylesheet named "style.css".

How to Use JavaScript in a Webpage

To link to an external JavaScript file, use the <script src="script.js"></script> element before the closing </body> tag. Example: 

  <script src="script.js"></script>
To write JavaScript directly within an HTML file, include the JavaScript code between <script></script> tags. Example: 
  <script>
  alert('Hello, world!');
  </script>
